# Medication_plan
针对有慢性病的老年人，如果用的药的类型比较多，哪个药少了买哪个药，比较麻烦，然后什么时候吃什么药有时候也搞错

**该程序实现两个功能：**
1. 根据天数计算每种药要买的数量及总价
2. 根据药品信息制定服药计划，如早中晚，饭前饭后

在执行完之后，会把结果保存到db.xlsx的两个sheet里，方便处理与打印
